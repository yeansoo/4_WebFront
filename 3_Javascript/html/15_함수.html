        function 함수명([매개변수]){ // 함수 선언 
            // 코드 == 함수 정의 
            
        }

        함수명([매개변수]); // 함수 호출

익명 함수(이름이 없는 함수)

    
        function([매개변수]){
            
        }
    

        - 이름이 없는 함수이기 때문에 필요 시 마음대로 호출하는 것이 불가능

        - 이벤트 핸들러와 같이 
            선언된 함수가 여러 곳에서 재사용 되는 것이 아닌
            특정 동작이나 상황에서만 수행되는 함수가 필요할 경우 사용한다.
